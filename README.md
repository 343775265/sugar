### 项目介绍
此项目是主要让我们在紧张的学习中有一个放松解压的机会。我们可以在线欣赏音乐并有两个简单的js解压小游戏供你娱乐（今后还会继续增加一些放松小游戏）。

### 主要实现功能:
<P>1、用户登录</p>
<P>2、用户注册</p>
<P>3、管理员系统</p>
<P>4、管理员可以查询、修改、删除用户信息</p>
<P>5、用户登录主界面功能为在线音乐</p>
<P>6、另附js娱乐小游戏</p>

### 项目特色和亮点
本项目导航栏用了bootstrp的前端框架是样式美观、代码简洁。
